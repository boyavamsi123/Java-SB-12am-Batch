Inheritance:-
- Aquiring the properties of one class to another class.
- Reusability

class A{                     // A super class, parent class, Base class

}
class B extends A{           // B sub class, child class, Derived class

}

Types of inheritance:-
--------------------
1.Single
2.Multiple
3.Hierarchecal
4.Multi Level

Single:-

A - Base
|
B - Derived

Multiple:-

A   B  - super class
  c    - sub class
  
  -> Interface

Hierarchecal:-

       A        - super class

    B  C  D     - sub class

Multi Level:-

A - super
|
B - intermidiate super class
|
C - sub class
